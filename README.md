# Platformer C++ SDL

Implementation of a simple 2D platform game using C++ 11 and SDL 2.0.

## Overview

This project is a single-player 2D platformer implemented in C++ using the SDL library. Inspired from the original Dangerous Dave DOS game.

- Compiler used: MinGW g++ 6.3.0
- C++ version: C++ 11

## Getting started

### Prerequisites

- C++ compiler (g++)
- SDL 2.0 library

### Building

1. Clone the repository
2. Run `build.bat`. The executable will be created in the `bin` folder.
3. The bin folder should contain `SDL2.dll`, `SDL2_ttf.dll` and `SDL2_image.dll` to run the compiled executable.
